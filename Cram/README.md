# Docs

## Emojis

- ⭐: Recommended
- ❌: Not recommended
- ⏺️: Default
- 🏷️: Cheap
- 💲: High price (cost) / Expensive
- ⚡: Low latency / High throughput (transactions)
- 🐌: High latency / Low throughput (transactions)
- 🎲: Random
- 💎: Premium only
- 🦺: Standard / Not premium
- 🏋🏿: Transaction-intensive
- 🙋‍♂️: High availability
